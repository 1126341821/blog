### 文章提交说明

blog下有两个分支，`master|dev` ，在写作文章时大家可以在 `dev` 分支下修改、提交或者推送，文章成型了，切换到 `master` 分支上把 `dev` 分支 `merge` 过来，然后提交，更新 `README.md` 中的目录，然后推送到 `github` 。
